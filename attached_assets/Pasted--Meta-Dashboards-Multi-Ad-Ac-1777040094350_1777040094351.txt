أنت خبير تحليل بيانات وإعلانات Meta، ومتخصص في بناء Dashboards احترافية متعددة الحسابات (Multi-Ad Accounts Dashboard) تعتمد على بيانات مباشرة من Meta Ads API.

سيتم تزويدك ببيانات Live من Meta API لعدة حسابات إعلانية، كل حساب يحتوي على:

* Campaigns / Ad Sets / Ads
* Spend / Impressions / Clicks / Purchases
* CTR / CPC / CPM / CPA
* Conversion Rate

🎯 الهدف:
بناء Dashboard احترافي يحتوي على Tabs، بحيث:

* كل حساب إعلاني يكون له Tab منفصل
* المستخدم يقدر يتنقل بين الحسابات بسهولة
* كل Tab يعرض Overview كامل للحساب الخاص به

━━━━━━━━━━━━━━━━━━━
🧱 أولًا: هيكل النظام (Multi Accounts Structure)

* أنشئ Tabs في أعلى الصفحة

* كل Tab يمثل:
  👉 حساب إعلاني واحد (Ad Account)

* عند اختيار Tab:
  👉 يتم تحميل Dashboard خاص بهذا الحساب فقط

* لا تخلط البيانات بين الحسابات

━━━━━━━━━━━━━━━━━━━
📊 ثانيًا: معالجة البيانات

لكل حساب إعلاني:

* احسب:
  CTR / CPC / CPM / CPA / Conversion Rate
* اعرض:

  * الأداء الحالي
  * مقارنة بالفترة السابقة (7 أو 14 يوم)

━━━━━━━━━━━━━━━━━━━
🔝 ثالثًا: KPI Bar (لكل حساب)

داخل كل Tab اعرض:

* 💰 Spend
* 🛒 Purchases
* 📉 CPA
* 👆 CTR
* 📊 CPM
* 🔄 Conversion Rate

✔️ مع مقارنة:

* ↑ أخضر = تحسن
* ↓ أحمر = تدهور

━━━━━━━━━━━━━━━━━━━
🚨 رابعًا: Health Status (لكل حساب)

* 🟢 الحساب شغال كويس
* 🟡 يحتاج تحسين
* 🔴 خطر

بناءً على:

* CPA
* Trend الأداء
* نسبة الحملات الخاسرة

━━━━━━━━━━━━━━━━━━━
🧠 خامسًا: Priority Engine (لكل حساب)

اعرض أهم 3 قرارات لكل حساب:

مثال:

1. ❌ أوقف Campaign X
2. 🔥 زوّد Budget Campaign Y
3. 🟡 حسّن Creative Campaign Z

━━━━━━━━━━━━━━━━━━━
📊 سادسًا: Top & Worst Campaigns

🟢 أفضل الحملات:

* CPA منخفض / أداء قوي

🔴 أسوأ الحملات:

* CPA عالي / بدون نتائج

✔️ مع زر:

* "إيقاف سريع"

━━━━━━━━━━━━━━━━━━━
📈 سابعًا: Trend

* CPA آخر 7 أيام
* Orders آخر 7 أيام

━━━━━━━━━━━━━━━━━━━
🧨 ثامنًا: Alerts

* 🚨 CPA زاد
* ⚠️ Campaign خاسرة
* 🔥 فرصة Scale

━━━━━━━━━━━━━━━━━━━
📊 تاسعًا: Breakdown

* أفضل Creative
* أسوأ Creative
* أفضل Audience

━━━━━━━━━━━━━━━━━━━
💰 عاشرًا: Profit Insight

* "يتم إهدار X جنيه"
* "يمكن زيادة X طلب"

━━━━━━━━━━━━━━━━━━━
🎮 الحادي عشر: Quick Actions

* 🔥 Scale Winners
* ❌ Kill Losers
* ⚖️ Optimize CPA

━━━━━━━━━━━━━━━━━━━
🎨 UX

* Tabs واضحة لكل حساب
* اسم الحساب ظاهر بوضوح
* عربي بالكامل
* تصميم بسيط

━━━━━━━━━━━━━━━━━━━
⚠️ قواعد مهمة:

* كل Tab مستقل تمامًا
* لا يتم خلط البيانات بين الحسابات
* الهدف: المستخدم يفتح أي حساب → يفهمه خلال 10 ثواني

━━━━━━━━━━━━━━━━━━━

Dashboard متعدد الحسابات يحتوي على:

* Tabs لكل حساب إعلاني
* داخل كل Tab Overview كامل
* يساعد المستخدم يعرف:

  * الحساب ده بيكسب ولا بيخسر؟
  * أوقف إيه؟
  * أزود إيه؟
  * أصلّح إيه؟

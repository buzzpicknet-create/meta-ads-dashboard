أنت خبير في بناء Dashboards ذكية لتسويق الأداء، ومتخصص في تصميم تجربة مستخدم (UX) تربط بين التحليل واتخاذ القرار والتنفيذ.

🎯 الهدف:
إنشاء تجربة متكاملة تربط بين:

1. صفحة Overview
2. صفحة Campaigns
3. صفحة How To + Calculator

بحيث يتم تحويل المشكلة → إلى حل → إلى محاكاة مباشرة

━━━━━━━━━━━━━━━━━━━
🔗 أولًا: الربط بين الصفحات (Core UX Logic)

عند عرض أي مشكلة في:

* Overview
* Campaigns

يجب أن تكون:

👉 Clickable (قابلة للضغط)

━━━━━━━━━━━━━━━━━━━
🎯 عند الضغط على المشكلة:

مثال:

* "CTR منخفض"
* "CPA عالي"
* "Conversion Rate ضعيف"

👉 يتم توجيه المستخدم تلقائيًا إلى:

صفحة 👉 How To + Calculator

━━━━━━━━━━━━━━━━━━━
🧠 ثانيًا: Auto Diagnosis داخل صفحة How To

عند دخول المستخدم من المشكلة:

يجب أن:

1. يتم تحديد المشكلة تلقائيًا (Auto Selected)
2. يتم عرض القسم المناسب مباشرة

مثال:

لو ضغط على:
"CTR منخفض"

👉 يتم فتح:

* Section تحسين CTR
* بدون ما المستخدم يدور

━━━━━━━━━━━━━━━━━━━
📌 ثالثًا: عرض الحل مباشرة

اعرض:

* المشكلة
* السبب
* الحل (خطوات واضحة)

مثال:

"CTR منخفض → غيّر الكريتف + استخدم Hook أقوى"

━━━━━━━━━━━━━━━━━━━
🎮 رابعًا: ربط الحل بالـ Calculator

عند اختيار المشكلة:

👉 يتم تفعيل الحاسبة تلقائيًا

مثال:

لو المشكلة CTR:

* يتم تفعيل Slider:
  "تحسين CTR"

━━━━━━━━━━━━━━━━━━━
📊 خامسًا: عرض التأثير

اعرض مباشرة:

* CPA الجديد
* عدد الطلبات المتوقع
* الفرق (Delta)

━━━━━━━━━━━━━━━━━━━
🚨 سادسًا: Insight ذكي

اعرض جملة واضحة:

"لو حسّنت CTR بنسبة 20% → هتزود X طلب وتقلل CPA بنسبة X"

━━━━━━━━━━━━━━━━━━━
🎯 سابعًا: CTA قوي

زر واضح:

👉 "نفّذ الحل"

(حتى لو مجرد UI)

━━━━━━━━━━━━━━━━━━━
🎨 UX مهم جدًا:

* الانتقال يكون Smooth
* بدون تحميل صفحة جديدة (يفضل Modal أو Scroll)
* Highlight للمشكلة
* Auto Focus على الحل

━━━━━━━━━━━━━━━━━━━
🧠 ثامنًا: Priority Integration

لو المستخدم دخل من Overview:

👉 اعرض:

"دي أهم مشكلة عندك دلوقتي"

━━━━━━━━━━━━━━━━━━━
⚠️ قواعد:

* لا تجعل المستخدم يبحث
* كل شيء يتم تلقائيًا
* المشكلة → الحل → التأثير

━━━━━━━━━━━━━━━━━━━
🎯 النتيجة:

نظام ذكي يجعل المستخدم:

* يضغط على المشكلة
* يفهمها فورًا
* يرى الحل
* يرى تأثير الحل
* ينفذ القرار

بدون تفكير أو بحث
